Tabs

Для вкладок на всю ширину страницы - модификатор "fix-width" к элементу "header".

Для вкладок с фиксированной высотой, чтобы выравнивать вертикально иконки, контент или стрелки - модификатор "fix-height" к элементу "header".

Для вкладок с одной нижней линией - модификатор "line-bottom" к элементу "header".

Для вкладок со стрелками - модификатор "is-arrow" к элементу "header".

Для добавления фона к контенту - модификатор "fon" к элементу "content".

Для табов второго уровня - модификатор "sub" к элементу "content".

Для табов с расположением вкладок слева - модификатор "header-left" к блоку "tabs001".
