Выберите меню

Чтобы изменить разрешение, при котором меню перестраивается в гамбургер: в header-common.scss - $header-mobile-breakpoint: desktop/tablet/mobile, в header.js - var headerMobileBreakpoint = 1200/960/640

Выпадающие меню влево (если выпадающее меню вправо по ховеру невозможно) - модификатор "sub-menu--right" к блоку "sub-menu"

Меню в гамбургере справа - модификатор "mobile-right" к блоку "menu001"

Мега-меню - к элементу меню первого уровня добавить класс "sub-mega-item", к самому меню - класс "sub-menu--mega"

Если в мега-меню нужен скролл, то добавить класс "sub-menu--custom-scroll". Высота задается через переменную $sub-menu-custom-scroll (header--003)

Мега-меню с вложенностью - к элементу меню первого уровня добавить класс "sub-title-mega-item", к самому меню - класс "sub-menu--title-mega" (подробнее - header--003)

Акции в мега-меню - к элементу меню первого уровня добавить класс "sub-action" (header--003)

Мега-меню в виде табов - к элементу меню первого уровня добавить класс "sub-tabs-item", к самому меню - класс "sub-menu--tabs". К активному пункту меню добавить класс "sub-item--show" (header--004)

Пункты меню с image - добавить элементу меню класс "menu-photo-item" и новый элемент с классом "menu-photo-link" (header--004)

Если нужно разбить пункты меню по колонкам - обернуть элементы меню в "menu-column". Добавить отступы у колонок - "menu-column--padding" (header--004)

Поменять вид поиска: добавить модификатор "overmenu" ("mods": ["overmenu"]) к блоку "search" и к элементу "search-icon"
